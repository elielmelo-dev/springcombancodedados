Olá! Essa é a primeira atividade valendo nota!

Nesse projeto temos essa rota principal que descreve o projeto.
Alem da rota principal temos mais 3 rotas, que são:
1ª Rota - Calculadora de IMC
2ª Rota - Retorna o texto inserido na URL pelo usuário
3ª Rota - Retorna a médida das notas inseridas.

Nas rotas abaixo você deve substituir o que está entre as chaves { }, colocando os valores que deseja ou o seu texto


Rota principal: http://localhost:3500/

http://localhost:3500/imc/{peso}/{altura}

http://localhost:3500/texto/{texto}

http://localhost:3500/nota/{nota1}/{nota2}



Nesse projeto também temos a ligacao com o banco de dados,onde podemos inserir informações em formato JSON,por meio do Postmann,configurado como POST,usando o endereço:

http://localhost:8080/api/v1/usuarios

Preenchendo os dados JSON de envio,conforme abaixo:

{
"username":"email.qualquer",
"password":"senhaQualquer",
"role":"ROLE_ADMIN"
}
