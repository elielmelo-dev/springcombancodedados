Bem-vindo! Esta é a rota raiz.

Nesse projeto temos essa rota principal que descreve o projeto. Alem da rota principal temos mais 3 rotas,uma para calcular o IMC,outra para retornar qualquer texto inserido pelo usuario e a ultima que o usuario insere duas notas e elas são somadas.Nessa apresentação das rotas,onde tem as chaves, devemos inserir os valores sem elas. As mesmas são indicaçao que precisa de uma substituição pelos valores

Rota principal: http://localhost:8080/

http://localhost:8080/imc/{peso}/{altura}

http://localhost:8080/texto/{texto}

http://localhost:8080/nota/{nota1}/{nota2}

Nesse projeto também temos a ligacao com o banco de dados,onde podemos inserir informações em formato JSON,por meio do Postmann,configurado como POST,usando o endereço:

http://localhost:8080/api/v1/usuarios

Preenchendo os dados JSON de envio,conforme abaixo:

{
"username":"email.qualquer",
"password":"senhaQualquer",
"role":"ROLE_ADMIN"
}
